Train Tracked

An app and API which interface with the Rail Delivery Group's Darwin™ API to provide slightly better™ train tracking, created as a Level 5 project @ UoP. Will remain closed-source until after the moderation period (Aug 2024)

Homelab

Personal homelab, which currenly consists of 2 Lenovo 'Tiny' PCs (1x M700 & 1x M720q) running Proxmox. They run a few publically accessible services (this website, and a few of my other projects), as well as game servers and a media server

picOLED

An OLED display and RP2040-based microcontroller, which can be connected to a computer to display important stats. Written entirely in C++, the driver & device communicate using a serial protocol over USB (Github)

More Projects

Minima

Minima is a URL shortening API written in Dart. It's not overly complex but is a good introduction to creating APIs using Dart & Shelf (There's also a little bit of file I/O)

Source code available on Github (Last updated 2023-05-10)

TB-69

TB-69 is a 69% (nice) ISO hand-wired keyboard of my own design. The chassis was desgined using Fusion 360 and 3D printed. I am currently in the process of wiring the keyboard, which is proving to be more difficult than I was expecting due to the small size of connections and the inadequacy of my cheapo uncontrolled soldering iron

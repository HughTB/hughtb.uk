My Projects

Train Tracked

Train Tracked is a "better" UK train tracking app, created as part of a Level 5 Computer Science project at the University of Portsmouth

Developed using Dart, Flutter and the Rail Delivery Group's delightful Darwin API. As well as the app itself, I have also developed an API which sits between Darwin and the app, which handles converting an XML SOAP API to a much nicer JSON REST API, as well as some basic parsing

You can't view the source code on GitHub (Last updated 2024-01-15), since the marking and moderation period hasn't ended yet :P

picOLED

picOLED is a small USB serial display which can be connected to a computer to show important information, such as CPU or memory usage (More to come in the future!). It is written entirely in C++ and should run on any RP2040-based microcontroller with an I2C interface.

Currently, the device itself is fully functional, but waiting for a complete driver to be written. I am in the process of writing the Linux version of this driver, which has at the time of writing, support for CPU and Memory usage, CPU temperature (On certain platforms, namely Intel 8th Gen and AMD Zen 2 CPUs) and GPU temperature (On RDNA 2 AMD GPUs). A Windows driver may come into existence in the future, but this is a low priority to me as I do not anticipate anyone using this project other than myself, who uses Linux for my main desktop computer.

You can view the source code on GitHub (Last updated 2023-09-07)

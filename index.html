My Projects

PicOLED

PicOLED is a small USB/serial display which can be connected to a computer to show important information, such as CPU or memory usage (More to come in the future!). It is written entirely in C++ and should run on any RP2040-based microcontroller with an I2C interface.

Currently, the PicOLED device itself is fully functional, and just waiting for a complete driver to be developed. I am currently in the process of creating the Linux version of said driver, and so far it supports sending memory and CPU usage on all systems, and partial temperature information on specific systems (8th Gen Intel systems). Whether a Windows driver will be created remains to be seen (I'm scared of windows.h)

You can view the source code on GitHub (Last updated 2023-08-03)
